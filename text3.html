Version Control Options

  • Old skool: CVS/SVN (client/server)

  • New skool: Git, Bazaar, Mercurial (distributed)

  • Git is by far the most popular, and has become the industry standard for version control. Used for the Linux kernel

  • Mercurial is used by Python, and others

  • Bazaar is used by Ubuntu, and supported by Canonical
